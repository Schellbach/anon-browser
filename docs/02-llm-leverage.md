# LLM leverage (not the product)

Anon is a **privacy + Bitcoin** browser. Frontier models accelerate hard browser work.

## Build-time (primary)

- Filter / scareware rule proposals from redirect graphs
- Chromium/Brave rebase and compile-out patches
- Import parsers + Playwright suites
- Packaging/CI templates

## Runtime (optional, later)

- Local “is this scareware/phishing?” assist
- User-gated; disableable; fail closed
- Default: no page content to cloud

## Never

- Cloud LLM as required dependency for browsing
- Keys or seed material in any LLM context
- “AI browser” positioning in UI or marketing
